La funcion Cargar ingredientes le permite al usuario cargar informacion directamente en la base de datos, para realizar este proceso correctamente se debe seguir las siguientes recomendaciones:
  • El archivo debe ser tipo excel (xls, xlsx).

  • El archivo no debe tener celdas en blanco.

  • No debe haber dos registro con el id_insumo e id_producto iguales, esto no es permitido!

  • El orden las columnas debe corresponder con el siguiente:
    (id_producto, id_insumo, cantidad).

  • No insertar el nombre de las columnas, el contenido debe ser solo datos.

  • Todas las celdas deben estar en el formato 'General'.
